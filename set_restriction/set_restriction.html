Features to restrict:

Locked features are shown in red- they cannot be used. Any unlocked features can be restricted by checking the box next to them. See the Challenges button to unlock more features.

Built-ins to restrict:

Locked built-in functions are shown in red- they cannot be used. Any unlocked built-ins can be restricted by checking the box next to them. To unlock a built-in, click the Implement button next to it and write a working implementation in the code editor on the right.

Write Built-ins:

To be able to use built-in functions, you will need to implement them yourself first. Click on the "write" button next to the function you want to implement, write your version below and press the run button to test it. If your function passes all the tests, you are free to use it in your code!
